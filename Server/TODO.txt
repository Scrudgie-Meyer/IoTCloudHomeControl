


User register -> User login -> User logout

Device registration -> Device possible scenarios -> run scenario -> stop scenario

Possible scenarios:


1. Автоматичне вмикання світла за розкладом
✅ Пристрої: лише розумна лампа
О 7:00 ранку світло автоматично вмикається
О 23:00 світло автоматично вимикається
Якщо потрібно, можна змінювати розклад через додаток
💡 Код логіки:
Використання BackgroundService, який перевіряє час кожну хвилину.


2. Контроль відкриття дверей
✅ Пристрої: дверний сенсор
Якщо двері відчинені більше 30 секунд → надсилається повідомлення
Якщо двері відкриваються після 22:00 → надсилається тривожне сповіщення
💡 Код логіки:
Використання подій OnDoorOpen, OnDoorClose + таймер для затримки.


3. Сенсор руху для світла
✅ Пристрої: датчик руху + лампа
Коли виявлено рух → світло вмикається
Якщо руху немає 5 хвилин → світло вимикається
💡 Код логіки:
Перевірка статусу руху + таймер зворотного відліку.

4. Кнопка для екстреного виклику
✅ Пристрої: смарт-кнопка

Натискання кнопки → надсилається повідомлення рідним
Якщо натиснути двічі → викликається швидка допомога
💡 Код логіки:
Відстеження кліків протягом 3 секунд.

5. Автоматична вентиляція в кімнаті
✅ Пристрої: датчик температури + вентилятор

Якщо температура > 28°C → вентилятор вмикається
Якщо температура < 25°C → вентилятор вимикається
💡 Код логіки:
Перевірка температури кожні 30 секунд.


6. Лічильник води з попередженням
✅ Пристрої: розумний лічильник води
Якщо споживання перевищує норму на день → приходить повідомлення
Якщо є витік води (споживання різко збільшилося) → попередження + аварійне перекриття води (якщо є клапан)
💡 Код логіки:
Логіка аналізу змін об'єму води.


7. Віддалене керування обігрівачем
✅ Пристрої: розумна розетка
Якщо температура < 18°C, можна увімкнути обігрівач з телефона
Якщо виходиш з дому, можна його вимкнути віддалено
💡 Код логіки:
Додаток + API для керування реле.


8. Запис відео при русі біля дверей
✅ Пристрої: датчик руху + камера
Якщо біля дверей зафіксовано рух → запускається запис на 30 секунд
Якщо рух о 3:00 ночі → додаткове повідомлення на телефон
💡 Код логіки:
Запуск відеозапису при події OnMotionDetected
